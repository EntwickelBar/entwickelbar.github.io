Von den Teilnehmern

Session: Clojure & Beer

In der Session "Clojure & Beer" haben wir uns mit der Programmiersprache Clojure beschäftigt und eine kleine Webanwendung entwickelt, die Stück für Stück Wörter in einem Text abhängig von einem Slider durch das beer-Emoji ersetzt hat.

Den Code dazu gibt es hier: http://github.com/n2o/clojurebeer

Session: HTTP, HTML, and Web Security

In dieser Session ging es rund um das Thema HTTP und Sicherheit. Bekannte Probleme, wie XSS, CSRF-Protection etc. wurden angesprochen.

Die Folien aus dieser Session können hier abgerufen werden: https://speakerdeck.com/joyclark/http-html-and-web-security

Session: Frontend-Testing

In dieser Session haben wir Erfahrungen rund um Frontend- und End-to-End-Testing ausgetauscht. Anschließend haben wir uns Jest angeschaut und andere Technologien diskutiert.

Demo-Code aus der Session kann hier angeschaut werden: https://github.com/holgergp/mvnJest
